# 连续打卡日历 (Habit Tracker)

这是一个类似图片的打卡应用，支持 2016-2026 年份切换，点击圆圈即可打卡并切换颜色。

## 功能

- **年份切换**：支持 2016 年到 2026 年。
- **打卡交互**：点击圆圈切换颜色（红、黄、绿、蓝、紫、灰）。
- **数据持久化**：打卡记录保存在浏览器本地 (LocalStorage)，刷新不丢失。
- **视觉风格**：仿照实物打卡日历的圆点设计。

## 运行

1. 安装依赖：
   ```bash
   npm install
   ```

2. 启动开发服务器：
   ```bash
   npm run dev
   ```

3. 构建生产版本：
   ```bash
   npm run build
   ```
