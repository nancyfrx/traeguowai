# TraceGuowai 项目整合平台

这是一个多项目整合管理平台，汇集了各种 AI 工具、Web 应用、游戏和实验性项目。该平台支持多级菜单展示，能够自动识别并嵌入不同类型的 Web 项目（Vite, Static 等）。

## 项目分类

### 🛠️ AI 工具 (AI_TOOL)
- **智谱 AI 聊天 (chat-gml)**: 基于智谱 AI 的聊天机器人，支持多模态交互。
- **DeepSeek Chat**: 基于 DeepSeek 大模型的聊天界面，提供纯净的对话体验。

### 📱 移动应用 (APP)
- **QQ音乐**: 高仿 QQ 音乐移动端界面与播放功能。
- **抖音**: 仿抖音短视频交互体验，支持视频滚动与评论。
- **高德扫街**: 基于高德地图的项目，用于数据采集与区域排名展示。
- **微信 / 新版微信**: 微信社交平台的复刻版，包含聊天、朋友圈等核心功能。
- **博客 (boke)**: 简洁的个人博客系统，支持 Markdown 渲染。
- **艺术市场 (blog)**: **(最新添加)** 结合 NFT 概念的艺术品交易博客，支持商品详情查看与模拟支付流程。

### 🎮 游戏中心 (GAME)
- **俄罗斯方块**: 经典的消除游戏，支持多种难度。
- **坦克大战飞机**: 复古风格的射击游戏。
- **消消乐**: 益智类消除游戏，具有精美的动画效果。

### 📂 其他 (OTHER)
- **动物百科**: 展示各类动物信息的科普类页面。
- **中国地图**: 交互式中国地图，支持各省份数据联动展示。
- **万年历**: 提供农历、公历及节日查询的实用工具。
- **技术文档**: **(最新添加)** 汇总了平台内各项目的开发设计文档与技术方案。

## 平台功能
- **多级导航**: 自动生成的侧边栏菜单。
- **视图切换**: 支持 PC 端与移动端（模拟器）视图切换。
- **快速刷新**: 一键重载当前项目。
- **独立开启**: 支持在新标签页中独立运行选中的项目。

## 快速开始

### 本地开发环境启动
项目提供了全自动的启动与停止脚本，一键管理所有前后端服务：

1.  **启动所有项目**：
    ```bash
    ./start_local.sh
    ```
    *该脚本会自动检查端口占用，启动 2 个 Java 后端和 7 个 Vite 前端服务，并将日志记录在 `logs/` 目录下。*

2.  **停止所有项目**：
    ```bash
    ./stop_local.sh
    ```

3.  **访问平台**：
    启动后访问 [http://localhost:5173](http://localhost:5173) 即可进入项目管理入口。

## 部署说明
- **自动化部署 (服务器)**：推荐在服务器端运行 `./update_and_deploy.sh`，它会自动拉取最新代码并执行安装。
- **生产环境打包**：支持通过 `prepare_deploy.sh` 进行一键打包，并部署至阿里云服务器（详情见 [ALIYUN_DEPLOY_GUIDE.md](./ALIYUN_DEPLOY_GUIDE.md)）。
- **动态环境**：平台入口 `index.html` 已实现环境自适应，本地运行自动连接开发服务器，生产部署自动使用相对路径。
